## 1.cd
Most probably command which was used the most.It is used to change the diretory and is very important and might as well make or break your day.I had very fun using this command especially.
## 2.ls
Lists out the contents/files of the current directory.
## 3.cp
Let's you copy and paste a specified file to a certain destination.interestingly if there iisn't a prticluar file matching the name of the destination the cp command will make a copy of the file in the same directory with the file name of the destination.
## 4.mkdir
A very important command which allows the user to create a new directory.
## 5.rm 
Used to remove a certain file or folder.

## GIT COMMANDS USED
## 1.git clone
Used to clone a certian repository aka downloads it.
## 2.git status
Used to check the status of each file while making a commit or push, ie modifications,staged or staged etc.
## 3.git add
Used to add a file to the repository.
## 4.git commit 
Commits the changes and new additions to the repo and makes a log.(Along with commit message being added before pushing)
## 5.git push
Finally pushes the changes to the remote repository.(This is the case where I faced alot of errors while pushing the work into my remote repository)
## 6.git log
Used only in fourth challenge to check the commit messages provided in the repository and find the secret code.
